📋 View CSV generation prompts

PhonePe / GPay CSV Prompt: 

Generate a CSV file strictly in the following PhonePe / GPay transaction export format.
Output ONLY CSV. Do not add explanations.

Date,Description,Debit,Credit,Balance

RULES:
- Date format must be DD/MM/YYYY
- Description must be uppercase
- Debit = money spent, Credit = money received
- Only one of Debit or Credit should have value per row
- Amounts must be in decimal format (e.g., 250.00)
- Balance must be a running balance with decimals
- No negative values
- No extra spaces
- No extra columns
- No text outside CSV

HDFC / SBI Bank CSV Prompt: 

Generate a CSV file strictly in the following HDFC / SBI bank statement format.
Output ONLY CSV. Do not add explanations.

Value Date,Narration,Withdrawal Amt,Deposit Amt,Closing Balance

RULES:
- Date format must be DD-MM-YYYY
- Narration must follow bank-style text (UPI/, POS, NEFT, ATM, etc.)
- Withdrawal Amt = money spent
- Deposit Amt = money received
- Only one of Withdrawal or Deposit should have value per row
- Amounts must be in decimal format (e.g., 450.00)
- Closing Balance must be maintained realistically
- Some rows may have empty Closing Balance
- No negative values
- No extra spaces
- No extra columns
- No text outside CSV
